== Description ==

A responsive search widget with a search field that re-sizes in response to user screen size.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Drag the 'Responsive Search' widget into the sidebar.

== Frequently Asked Questions ==

= What does this plugin do? =

This plugin provides a search field that re-sizes to fit the screen size of the user.

== Screenshots ==

1. The Responsive Search widget looks just like the default WordPress search widget, but it adjusts to the size of the user's screen.
